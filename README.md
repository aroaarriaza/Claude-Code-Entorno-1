# Claude-Code-Entorno-1

Mi primer repositorio en GitHub, donde estoy aprendiendo Git, GitHub y
desarrollo web con la ayuda de Claude Code.

## La web en vivo

La página de este proyecto está publicada en:

**https://aroaarriaza.github.io/Claude-Code-Entorno-1/**

## Qué he aprendido en este proyecto

- Crear y configurar un repositorio en GitHub.
- El ciclo de trabajo con Git: editar, `add`, `commit`, `push`.
- Crear Pull Requests y fusionar ramas con `merge`.
- Entender qué es una rama (branch) y cómo trabajan en paralelo.
- Publicar una web en internet usando GitHub Pages.

## Tecnologías usadas

- HTML y CSS
- Un poquito de JavaScript (el contador de segundos)
- Git y GitHub
- GitHub Pages
